# OldMapleStory 楓之谷復古網站

## 主要特色
- 現代化UI設計 - 懷舊風格、圖鑑設計  
- JWT認證系統 - 安全的登入驗證
- 數據庫管理 - 實時性能監控
- 自動備份 - 數據庫備份恢復

## 技術棧
- Next.js 15, TypeScript, Tailwind CSS
- Prisma ORM + SQLite
- JWT認證

## 快速開始
```bash
bun install
bun run db:migrate
bun run db:seed  
bun run dev
```

## 管理後台
- 登入: /login
- 後台: /admin  
- 帳戶: admin@oldmaplestory.tw / admin123
